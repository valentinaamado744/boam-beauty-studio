# BOAM Beauty Studio

Sistema web de gestión de reservas para peluquería desarrollado con Flask, SQLite y Bootstrap.

## Funcionalidades

* Registro de reservas
* Visualización de reservas
* Eliminación de reservas
* Confirmación antes de eliminar
* Persistencia mediante SQLite

## Tecnologías utilizadas

* Python
* Flask
* SQLite
* Bootstrap 5
* HTML
* CSS

## Ejecución

Instalar dependencias:

```bash
pip install -r requirements.txt
```

Ejecutar aplicación:

```bash
python app.py
```

Abrir en navegador:

```text
http://127.0.0.1:5000
```
